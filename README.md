## Introduction

ST Monitor dedicates to provide a portable and enjoyable way for everyone to check the change of <b>Chinese</b> stock market, especially, by treemap. Each stock is a cube, the proportion represents its total market capitalization, the color indicates a specific index value such as its value change. All Information will be shown on the cube literally. And it is totally ad-free.

## Feature

### Fliter options

<b>SEVEN</b> options for fliter, including mainstream boards in mainland China. 

|  English Name  |  Chinese Name  |
|  ----  | ----  |
| Total A Shares | 全部A股 |
| SSE A Shares | 上证A股 |
| SZSE A Shares | 深证A股 |
| SSE Main Board | 上交所主板 |
| SZSE Main Board | 深交所主板 |
| SME Board | 中小板 |
| GEM | 创业板 |

### Multiple indicators

<b>SIXTEEN</b> indicators for checking. 

|  English Name  |  Chinese Name  |
|  ----  | ----  |
| 1 Day Performance | 涨跌幅 |
| 1 Week Performance | 近1周涨跌幅 |
| 1 Month Performance | 近1月涨跌幅 |
| 3 Months Performance | 近3月涨跌幅 |
| 6 Months Performance | 近6月涨跌幅 |
| 1 Year Performance | 近1年涨跌幅 |
| Year to Date Performance | 今年以来涨跌幅 |
| Relative Volume | 相对成交量 |
| PEG | PEG |
| P/S | 市销率 |
| P/B | 市净率 |
| Dividend Yield | 股息率 |
| PE Ratio (TTM) | 市盈率(TTM) |
| Forward P/E | 预测市盈率 |
| EPS Growth Past 5 Years | 过去5年EPS增长率 |
| Earnings Date | 业绩公布日 |
