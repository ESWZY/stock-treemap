## Introduction

ST Monitor dedicates to provide a portable and enjoyable way for everyone to check the change of <b>Chinese</b> stock market, especially, by treemap. Each stock is a cube, the proportion represents its total market capitalization, the color indicates a specific index value such as its value change. All Information will be shown on the cube literally. And it is totally ad-free.
